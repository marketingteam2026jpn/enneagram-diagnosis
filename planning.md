## 4. 心理学的設計（Psychology Agent）

### 4-1. 理論的基盤

エニアグラムは、人間の性格を9つの基本タイプに分類する性格類型論。Don Richard Riso & Russ Hudsonの発達レベル理論、Claudio Naranjoの性格構造分析、Sandra Maithriのスピリチュアルアプローチを統合的に参照。

**3つのセンター（知性の中心）:**

| センター | タイプ | 中心テーマ | 中心感情 |
|---|---|---|---|
| ガット（本能） | 8, 9, 1 | 自律性・境界線 | 怒り |
| ハート（感情） | 2, 3, 4 | アイデンティティ・自己価値 | 恥 |
| ヘッド（思考） | 5, 6, 7 | 安全・確実性 | 恐れ |

### 4-2. 質問設計原則

1. **社会的望ましさバイアスの低減**: 「〜すべきだ」型ではなく「〜することが多い」型を採用
2. **中央回避バイアス対策**: 7段階スケールで微妙な差異を捉える
3. **黙従バイアス対策**: 逆転項目を各タイプ1問ずつ配置（計9問）
4. **文脈効果の排除**: 質問は3センターを横断してシャッフル配置
5. **日本語最適化**: 10代後半〜30代の日常語彙。堅すぎず砕けすぎない表現

### 4-5. スコアリングアルゴリズム（高精度版）

```javascript
// ===== STEP 1: 基本スコア計算 =====
// 各タイプの6問の回答値を重み付き加算
// 逆転項目: score = (8 - response) × weight
// 通常項目: score = response × weight

// ===== STEP 2: 正規化 =====
// 各タイプのスコアを0-100に正規化
// normalizedScore = ((rawScore - minPossible) / (maxPossible - minPossible)) × 100

// ===== STEP 3: 応答バイアス補正（ipsative correction） =====
// 個人の平均回答傾向を補正
// correctedScore = normalizedScore - individualMean + 50

// ===== STEP 4: メインタイプ判定 =====
// 補正後スコアが最高のタイプ = メインタイプ

// ===== STEP 5: ウイング判定 =====
// メインタイプの隣接2タイプのうち、スコアが高い方
// 環状配列: 1-2-3-4-5-6-7-8-9-1
// 例: メインType 9 → Type 8 vs Type 1 → 高い方がウイング

// ===== STEP 6: トライタイプ判定 =====
// ガットセンター(8,9,1)から最高スコアのタイプ
// ハートセンター(2,3,4)から最高スコアのタイプ
// ヘッドセンター(5,6,7)から最高スコアのタイプ
// 3つをスコア順に並べる（例: 469, 853）

// ===== STEP 7: 信頼度スコア =====
// topScore - secondScore の差分で判定
// 差分15以上 → 信頼度95%
// 差分10-14 → 信頼度85%
// 差分5-9  → 信頼度70%
// 差分3-4  → 信頼度50%
// 差分0-2  → 信頼度30%（要再検討）

// ===== STEP 8: 成長・ストレス方向 =====
// 成長方向（統合）: 1→7, 2→4, 3→6, 4→1, 5→8, 6→9, 7→5, 8→2, 9→3
// ストレス方向（分裂）: 1→4, 2→8, 3→9, 4→2, 5→7, 6→3, 7→1, 8→5, 9→6
```
